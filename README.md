# WebDev
Start from Bitcamp





Some explaination 
-bin/www 项目启动必备文件
-configs 存放配置文件
-controllers 控制层
-libs 存放一些工具函数js等
-locales 存放i18n国际化json
-logs 存放错误日志
-middlewares 存放中间件
-models 存放数据库文件
-public 存放静态文件，如样式、图片等（由于咱们主要使用express书写后端api 所以该文件夹仅作为备用）
-routes 存放路由文件
-services 存放服务文件
-views 存放模板文件（由于咱们主要使用express书写后端api 所以该文件夹仅作为备用）
-app.js 主入口文件
-package.json 存储项目名、描述、作者、依赖等等信息
-readme.md 项目介绍等
